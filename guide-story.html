Dungeon unlock order, miasma levels, and how to access all areas of the game.

1 First Grassy Knoll — 5F [25min]
2 The Lost Forest — 10F [1h40min]
3 Deep Cave — 10F [3h20min]
4 Twilight Forest — 4F [2h00min]
5 The Secret Passage — 15F [3h45min]
6 Old Battlefield of Fog — 6F [2h00min]
7 Giant Tower of Justice — 30F [10h00min]
8 The Door to Another World — Dimensional Rift 9F [3h00min]
9 The Magic Castle of the End — 20F [7h00min]

The Illusory Royal Tomb

Unlocks when you gain access to M1. You start with floors 1-50. Each time you clear the final floor, you gain access to the next set of 50.

Loot schedule (repeating every 50 floors):

Every 50n+10, 50n+30: Rosetta Stone (+10 analysis)

Every 50n+20: HardenerZG (+50 upgrade value)

Every 50th floor: Black Key (for Fallen Junkyard)

Every 300th floor: Illusory Ore Whetstone (+10 corrosion)

Fallen Junkyard

After obtaining the Core Fragment (M2 unlocked), obtain a Black Key from The Illusory Royal Tomb. Keys drop every 50th floor.

See the Bosses page for detailed boss info.

The Door to Another World

Can lead to several dungeons depending on what code you enter. Some dungeons are the only source of powerful equipment.

To progress the story, you need to enter the code:

Reveal story code (spoiler) wonderland

If you want to find it yourself, explore the previous dungeons again. Check the end of the dungeon log for clues.

The Door also accesses semi-random dungeons. Entering any other code generates a random dungeon. Each code maps to exactly 1 dungeon. The game saves the 10 most recent codes.

See the Dungeons page for recommended keys.

Alchemist's Island

Accessed with a Map Fragment from the daily bonus chest. Three floors, each with specific materials:

1F [50min]: G-level hardeners

2F [50min]: All essences (silver, gold, star)

3F [50min]: Illusory Ore Whetstone only

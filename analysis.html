How Analysis Works

Analyzing equipment unlocks bonuses at each level. Higher analysis levels require higher corrosion levels on the item. Each equipment type (Weapon, Armor, Ring) gains different bonuses at each level.

Base analysis limit is 15 (from The Lost Nation's Magnifying Glass). Panoptes' Eye proof of subjugation increases the limit by +1 per level (max Lv10 = Lv25 analysis).

Key Milestones

Level 5: First upgrade limit increase (+100). Ring gets specialization boost.

Level 7: +1000 upgrade limit. Ring unique effect unlocked.

Level 9: Weapon set bonus (x1.5 STR). Requires Corrosion 200.

Level 13: Second weapon set bonus (x1.5 STR). Armor HP boost. Requires Corrosion 400.

Level 20: +19000 upgrade limit. Requires Corrosion 2000.
